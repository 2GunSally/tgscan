## 概览

该项目通过直观的界面简化了 Telegram 搜索，让用户可以快速查找频道、群组和聊天记录。

## 特点
- 直观的界面，便于搜索
- 快速查找频道、群组和聊天记录
- 近实时索引，以便更快速地搜索
- 支持聊天记录搜索
